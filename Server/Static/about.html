About us

Crypto Quote is a Full Stack course project made by George Brown Students. This project aims to show the students skills to develop a full application from front to back end, manipulating data and server services. This project is a website that provides some online crypto quotes and for register users, it simulates a Ether Walllet, managing users crypto balance and some crypto interactions.
Full Stack Development 2 is a course focused on more advanced JavaScript fundamentals, Node.js and its core modules, client-server models, and database interaction. Students will learn how to import additional modules and configure the project using npm. Express.js will be used to set up a web server, and students will learn how to interact with the Firebase Realtime Database using JavaScript and Node.js.
